# *peromyscus*

this is a mouse firmware produced for ECE 4760, in fall of 2025.

it links to:
- the pico sdk, https://github.com/raspberrypi/pico-sdk, BSD 3 clause
- equeue, https://github.com/geky/equeue/, MIT
- tinyusb, https://github.com/hathach/tinyusb/, MIT

these libraries *should* be AGPLv3 compatible.
