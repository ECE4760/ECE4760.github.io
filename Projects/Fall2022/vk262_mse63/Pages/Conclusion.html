Conclusion

Design Analysis

The final design was largely in line with expectations. When the members first proposed the project, they were unaware of pitch detection, and only later learned of the use of the cepstrum to calculate it. There aren't any legal standards that this design must conform to, so it isn't in violation of anything


If there were future development, notes of longer length which require a sustained output would be an interesting feature to add. Additionally, a penalty could be added for playing notes which aren't meant to be played. This would be beneficial to the gameplay experience, as someone can currently "pass" the game by playing many notes at once. Chord detection would be difficult to add but perhaps possible given enough time and research.


Intellectual Property Considerations

The only code not written by the members of the group are the FFT and VGA library examples provided by V. Hunter Adams. All other code is original. No proprietary parts or software were used in this project.


All songs included in the project are in the public domain.


There likely aren't patent opportunities for this project.
